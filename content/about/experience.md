## Senior SRE | NBCUniversal

**AWS | Serverless | Event-Driven | Security Operations**

Joined as a mid-level SRE. Promoted to Senior to help lead service and infra technical direction.

* **Event ETL Pipeline:** Designed and lead a project to run a high-throughput pipeline moving security events from Kafka. Met strict requirements for zero-loss data ingestion and low-latency analysis.
* **Perpetual Visibility Tooling:** Building and maintining services to track cloud assets across the service lifecycle, ensuring assets and changes are tracked for security and compliance.
* **Pentest Self-Service Platform:** Developed a restricted, secure platform allowing pentesters to provision their own access into cloud environments within guardrails.
* **Legacy Secret Sharing App Update:** Parched updated and migrated an old secret sharing application to ECS, making changes to fix security issues I found, improving code reliability, and making highly available.
* **Ownership:** Taking ownership of our services and supporting tech/setup.
